## 1. 개발 환경
- **프로젝트 개발 환경** : IntelliJ
- **Backend**: Java 17, Spring Boot 3.x
- **Database**: PostgreSQL
- **Build Tool**: Gradle
- **API 설계**: RESTful API
- **서비스 탐색**: Spring Cloud Eureka
- **API Gateway**: Spring Cloud Gateway
- **보안**: Spring Security, JWT, BCrypt
- **버전 관리** : Git
- **협업 툴** : Slack, Notion
